keypoints2body Documentation
=========================

.. image:: assets/header.svg
   :alt: keypoints2body
   :width: 100%

`keypoints2body` is a Python library for fitting SMPL-family body models (SMPL/SMPLH/SMPLX)
to 3D joints from single frames or motion sequences.
